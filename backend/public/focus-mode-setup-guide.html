Why You Need a Browser Extension

Due to browser security restrictions, websites cannot block other websites directly. To enable real website blocking during your focus sessions, you need to install a browser extension.

‚úÖ Good News: Once installed, the extension will block websites even when the Focus Mode page is closed!

Installation Steps

Step 1: Open Edge Add-ons Store

Click the button below to open the Microsoft Edge Add-ons store:

Open Edge Add-ons Store

Step 2: Search for Extension

In the search bar at the top, type one of these:

  • "BlockSite"
  • "StayFocusd"
  • "Block Site"

Step 3: Install Extension

Click the "Get" button on the extension you want to install.

When prompted, click "Add extension" to confirm.

Step 4: Configure Blocked Sites

After installation:

  • Click the extension icon in your browser toolbar (puzzle piece icon)
  • Add the websites you want to block (Instagram, Facebook, etc.)
  • Set up blocking schedules if desired

Step 5: Test It Out

Try visiting a blocked website to confirm it's working!

üí° Tip: Most extensions allow you to set a password so you can't easily disable blocking during focus time.

Pro Tips

  • Schedule blocking: Set specific times when sites should be blocked (e.g., 9 AM - 5 PM on weekdays)
  • Use categories: Block entire categories like "Social Media" instead of individual sites
  • Set a password: Prevent yourself from disabling the extension during weak moments
  • Whitelist important sites: Make sure work-related sites aren't accidentally blocked
  • Combine with Focus Mode: Use the timer in Focus Mode alongside the extension for maximum productivity
