### How to Enable Website Blocking

1. **Visit Focus Mode**
   - Go to `http://localhost:3000/focus-mode.html`

2. **Click "Get Extension"**
   - Green banner at top of page
   - Opens Edge Add-ons store

3. **Search for Extension**
   - Type "BlockSite" or "Block Site"
   - Or try "StayFocusd"

4. **Install & Configure**
   - Click "Get" to install
   - Add websites to block (Instagram, Facebook, etc.)
   - Set up schedules if desired

5. **Done!**
   - Extension blocks sites everywhere
   - Works even when Focus Mode is closed

### Need Help?
- Click "View Setup Guide" in the banner
